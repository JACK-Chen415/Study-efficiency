# PPT 图像生成素材包

本文件夹用于把当前项目汇报 PPT 交给 GPT image 模型或其他图像模型重新生成页面。

## 文件结构

- `PPT逐页大纲_给GPT_image模型.md`：11 页逐页大纲、版式建议和生成提示词。
- `slide-01_assets/`：第 1 页标题页使用的真实手机界面截图。
- `slide-08_assets/`：第 8 页系统界面展示页使用的真实系统截图。

## 使用建议

1. 先把 `PPT逐页大纲_给GPT_image模型.md` 发给图像模型。
2. 对第 1 页和第 8 页，附上对应页素材文件夹里的图片。
3. 其他页面不需要额外图片，按大纲生成流程图、信息卡和图表即可。
4. 所有数据口径必须保持：真实 completed=17，mock=120；mock 结果只用于演示链路，不代表真实学习规律。
